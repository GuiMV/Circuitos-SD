# Projeto de Lógica Digital - CircuitVerse

Este repositório contém projetos desenvolvido na plataforma [CircuitVerse](https://circuitverse.org/).

## 📁 Conteúdo — Arquivos compactados de projetos do CircuitVerse.

- `Contador_Gray.cv`
- Link: [https://circuitverse.org/users/318297/projects/contador-gray-240cd807-e467-4d7e-a473-896bb33fa42b](https://circuitverse.org/users/318297/projects/contador-gray-240cd807-e467-4d7e-a473-896bb33fa42b)
 <br>Descrição: Implementação de um contador utilizando o código Gray, em que apenas um bit muda a cada transição de estado.

- `Flip-Flops_RS_D_Contador.cv`
- Link: [https://circuitverse.org/users/318297/projects/flip-flops-11950002-5a38-4e63-9a3a-56a916489969](https://circuitverse.org/users/318297/projects/flip-flops-11950002-5a38-4e63-9a3a-56a916489969)
<br>Descrição: Projetos envolvendo flip-flops RS e D, além da implementação de um contador sequencial.

- `BCD-Display_de_7_Segmentos.cv`
- Link: [https://circuitverse.org/users/318297/projects/bcd-gt-display-de-7-segmentos](https://circuitverse.org/users/318297/projects/bcd-gt-display-de-7-segmentos)
<br>Descrição: Conversor de código BCD para display de 7 segmentos, responsável por decodificar e exibir dígitos decimais.

- `Contadores Síncronos e Assíncronos`
- Link: [https://circuitverse.org/users/318297/projects/contadores-sincronos-e-assincronos-3a1196d6-6492-46cb-8609-6c3ed693bd78](https://circuitverse.org/users/318297/projects/contadores-sincronos-e-assincronos-3a1196d6-6492-46cb-8609-6c3ed693bd78)

- `Contador Up-Down`
- Link: [https://circuitverse.org/users/318297/projects/contador-up-down-33936834-91ee-4476-adb7-8a70a35d94b8](https://circuitverse.org/users/318297/projects/contador-up-down-33936834-91ee-4476-adb7-8a70a35d94b8)

## 📥 Como abrir o Arquivo

- Acesse o Link ou baixe o arquivo e siga os passos:
1. Vá para o simulador [CircuitVerse Simulator](https://circuitverse.org/simulator)
2. Crie um novo projeto.
3. Use a opção **Import from File** e envie o `.cv`.
